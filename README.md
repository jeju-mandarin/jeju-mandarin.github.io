# 🚀 jeju-mandarin의 유틸리티 허브 (https://jeju-mandarin.github.io)

일상 속에서 유용하게 사용할 수 있는 간단한 정적 웹 도구(유틸리티)들을 모아둔 메인 포털입니다.

이 저장소는 `jeju-mandarin.github.io`의 메인 페이지를 호스팅하며, GitHub Pages를 통해 배포된 다른 프로젝트들로 연결하는 대문 역할을 합니다.

🔗 **[메인 허브 바로가기](https://jeju-mandarin.github.io)**

---

## 🛠️ 포함된 프로젝트

현재 다음 유틸리티들을 이용할 수 있습니다.

1.  **[🍀 로또 번호 생성기](https://jeju-mandarin.github.io/lotto/)**
    * 원하는 개수만큼 중복되지 않는 로또 번호 조합을 생성합니다.

2.  **[💰 예적금 만능 이자 계산기](https://jeju-mandarin.github.io/savecalc/)**
    * 복리/단리, 예금/적금 4가지 방식의 세후 이자 및 차액을 비교 계산합니다.

3.  **[⏳ D-Day 계산기](https://jeju-mandarin.github.io/d-day/)**
    * 기념일, 시험일 등 특정 날짜까지 남은/지난 날짜를 계산하고 관리합니다.

4.  **[⌨️ 실시간 글자 수 세기](https://jeju-mandarin.github.io/text-utils/)**
    * 글자 수, 바이트(UTF-8), 단어 수, 예상 읽기 시간 및 실제 입력 시간을 실시간으로 측정합니다.

## 💻 사용 기술

모든 프로젝트는 백엔드 서버 없이 순수한 프론트엔드 기술로 구현되었습니다.

* HTML5
* CSS3
* Vanilla JavaScript (ES6+)
